# Kace Kodi Repository

Kodi Repository mit xShip und OSMC Skin.

## Installation in Kodi

### Methode 1: Repository installieren (empfohlen)

1. Lade `repository.kace-1.0.0.zip` herunter:
   - [Download](https://github.com/kace-dev/kodi/raw/main/repo/repository.kace/repository.kace-1.0.0.zip)

2. In Kodi:
   - **Addons** → **Addon-Browser** (Paket-Symbol oben links)
   - **Aus ZIP-Datei installieren** → `repository.kace-1.0.0.zip` auswählen
   - **Aus Repository installieren** → **Kace Repository**
   - Gewünschtes Addon auswählen und installieren

### Methode 2: Direkt als ZIP installieren

1. Lade die gewünschte Addon-ZIP herunter:
   - [plugin.video.xship](https://github.com/kace-dev/kodi/raw/main/repo/plugin.video.xship/plugin.video.xship-2026.01.18.zip)
   - [skin.osmc](https://github.com/kace-dev/kodi/raw/main/repo/skin.osmc/skin.osmc-21.2.2.zip)

2. In Kodi: **Addons** → **Aus ZIP-Datei installieren** → ZIP auswählen

## Enthaltene Addons

| Addon | Version | Beschreibung |
|-------|---------|--------------|
| plugin.video.xship | 2026.01.18 | Streaming Addon mit s.to Scraper |
| skin.osmc | 21.2.2 | OSMC Skin für Kodi |

## Repo aktualisieren

Nach Änderungen an Addons einfach `UPDATE_REPO.bat` ausführen, dann committen und pushen.
